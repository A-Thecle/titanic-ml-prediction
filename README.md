# Titanic Survival Prediction

## Project Overview
This project uses Machine Learning to predict whether a passenger survived the Titanic disaster.

## Dataset
Titanic Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Machine Learning Model
- Logistic Regression
- Neural Network

## Machine Learning Models Tested

Several machine learning models were tested and compared:

- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- XGBoost
- AdaBoost

## Final Model Selection

After comparing the performance of the different models, the **AdaBoost classifier** was selected.

The model was optimized using **GridSearchCV** to find the best hyperparameters.

## Model Performance

The AdaBoost model showed good performance on both:
- Training data
- Test data

he results indicate that the model **generalizes well** and does **not suffer from overfitting**.
